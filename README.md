# FiveM Dashcam
> A FiveM server-sided script that allows a dashcam view for police vehicles

Images:

![](dashcam.png)

## Installation

Windows:

1. Download it
2. Extract it to your resources folder
3. Add it to server cfg
4. Done

## Controls

Press C to enable the dashcam and disable the dashcam
